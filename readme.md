# shelljs-async

bash syntax and gnu coreutils in javascript

async version of [shelljs](https://github.com/shelljs/shelljs)

- works in a web browser
  - browserfs
  - xterm.js
- async by default
- emulate an interactive shell
  - parse input strings from a bash prompt
